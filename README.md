# 🚴 Bicycle Sales and Profit Analysis Dashboard in Excel

## 📌 Project Overview
This project is an interactive Excel dashboard designed to analyze bicycle sales data across three key dimensions: Segment, Country, and Product.

The dashboard provides a complete view of business performance using key metrics such as Profit, Sales, Orders, and Units Sold, with year-wise comparison between 2013 and 2014.

---

## 📊 Dashboard Pages

### 🔹 Segment Analysis Dashboard
The Segment Dashboard provides insights into business performance across different customer segments.

**Segments Included:**
- Channel Partners  
- Enterprise  
- Government  
- Midmarket  
- Small Business  

**Key Visuals:**
- Segment-wise Profit comparison (2013 vs 2014)
- Segment-wise Sales distribution
- Orders distribution using Pie Chart
- Units Sold comparison

**Insights:**
- Government segment generates the highest profit and unit sales
- Small Business shows strong growth in 2014
- Enterprise segment has lower contribution compared to others

---

### 🌍 Country Analysis Dashboard
The Country Dashboard analyzes performance across different geographical regions.

**Countries Included:**
- Canada  
- France  
- Germany  
- Mexico  
- United States of America  

**Key Visuals:**
- Radar chart for Units Sold comparison
- Line chart for Profit trends (2013 vs 2014)
- Bar chart for Sales comparison
- Pie chart for Orders distribution

**Insights:**
- France and Germany contribute significantly to profit
- Sales increased in 2014 compared to 2013
- Orders are evenly distributed across countries

---

### 🛍️ Product Analysis Dashboard
The Product Dashboard focuses on performance across different bicycle products.

**Products Included:**
- Amarilla  
- Carretera  
- Montana  
- Paseo  
- Velo  
- VTT  

**Key Visuals:**
- Product-wise Profit comparison
- Product-wise Sales distribution
- Units Sold comparison
- Orders distribution using Pie Chart

**Insights:**
- Paseo is the top-performing product in sales and units sold
- VTT and Velo show consistent performance
- Product demand varies across categories

---

## ⚙️ Features
- Interactive slicers for:
  - Year (2013, 2014)
  - Month / Quarter
  - Discount Band (High, Low)
- Dynamic charts (Bar, Pie, Line, Radar)
- Navigation buttons between dashboards
- Clean and visually appealing design
- Fully interactive and user-driven analysis

---

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers & Timeline
- Conditional Formatting

---

## 🚀 How to Use
1. Download the Excel dashboard file
2. Open in Microsoft Excel (2016 or later recommended)
3. Use slicers to filter the data
4. Navigate between dashboards using buttons
