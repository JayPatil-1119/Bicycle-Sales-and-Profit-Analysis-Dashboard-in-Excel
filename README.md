# 🚴 Bicycle Sales and Profit Analysis Dashboard in Excel

## 📌 Project Description
This project is an interactive Excel dashboard designed to analyze bicycle sales performance across different dimensions such as Segment, Country, and Product.

The dashboard provides insights into key business metrics including Sales, Profit, Orders, and Units Sold, helping users make data-driven decisions.

---

## 🎯 Objectives
- Analyze sales and profit trends
- Identify top-performing segments and products
- Compare performance across countries
- Build an interactive dashboard using Excel

---

## 📊 Dashboard Overview

### 🔹 Segment Analysis
- Segment-wise Profit, Sales, Orders, and Units Sold
- Comparison between 2013 and 2014

### 🌍 Country Analysis
- Country-wise Sales, Profit, and Orders
- Includes Radar, Line, and Bar charts

### 🛍️ Product Analysis
- Product-wise Sales, Profit, Units Sold, and Orders
- Products: Amarilla, Carretera, Montana, Paseo, Velo, VTT

---

## ⚙️ Features
- Interactive slicers (Year, Month, Quarter, Discount Band)
- Dynamic charts (Bar, Pie, Line, Radar)
- Navigation buttons between dashboards
- Clean and user-friendly design

---

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers and Timeline
- Conditional Formatting

---

## 📈 Key Insights
- Government segment generates the highest profit
- Paseo is the top-performing product
- France and Germany contribute significantly to sales
- Sales increased from 2013 to 2014
- Discount impacts purchasing behavior

4. Navigate between dashboards using buttons
